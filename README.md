# NextAura

Public proof for NextAura's direction.

This repository is the public confidence layer for the company:
- website and company narrative
- high-level product direction
- public architecture posture
- safe proof artifacts and inspectable signals

It is not the full engine room.

NextAura follows a hybrid posture:
- public proof
- private engine

The private stack contains the compounding operational system:
- vault and credential-handling internals
- workflow gating and provider-approval logic
- ROI models and scaffold composition
- memory tuning, runbooks, and operational intelligence

The goal is to win through execution speed and trust while keeping the
compounding parts of the stack private.
